
# Landing Page 
- NextJS 13
- Tailwind
- Framer motion
- Lottie





## Installation

Install with npm

```bash
  npm install --force

  npm run dev

  npm run build
  npm run start (Production)

  #or 
  yarn install --force
```
    
## Related Coming Soon Modal

- Remove data in LocalStorage to show Modal Coming Soon
- set countTime
+ countTime > 0: modal will be show
+ countTime = -1: modal never show again
